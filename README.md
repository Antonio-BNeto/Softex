# 🚀 Softex – Back End com Python e Django  

## 📌 Sobre o Repositório  
Este repositório foi criado para armazenar exemplos, exercícios e atividades desenvolvidos durante as aulas do curso **Back End com Python e Django** da **Softex**.  

Aqui você encontrará:  
- Códigos de exemplo apresentados em aula;  
- Exercícios e desafios práticos;  
- Projetos desenvolvidos ao longo do curso.  

---

## 📚 Estrutura do Repositório  
- **/exemplos** → Códigos de exemplo apresentados em aula.  
- **/atividades** → Exercícios e desafios propostos para prática.  
- **/projetos** → Projetos práticos desenvolvidos nos módulos.   

---

## 🎯 Objetivo do Curso  
O curso tem como objetivo desenvolver as habilidades necessárias em **programação back-end**, com foco em **Python, Django e boas práticas de desenvolvimento de software**, para que o aluno seja capaz de projetar soluções web **seguras, robustas e escaláveis**.

## ✅ Checklist de Módulos  

### **Módulo 1 – Fundamentos**  
- ✅ Lógica de Programação com Python  
- ✅ Git, GitHub e GitLab  
- 🔄 Engenharia de Software  

### **Módulo 2 – Python e Orientação a Objetos**  
- ⏳ Classes, Objetos, Encapsulamento e Polimorfismo  
- ⏳ Análise e Projeto Orientado a Objetos  
- ⏳ Testes de Software  

### **Módulo 3 – Banco de Dados**  
- ⏳ Modelagem Conceitual, Lógica e Física  
- ⏳ SQL (consultas, joins, agregações)  
- ⏳ Normalização e Chaves (PK, FK)  

### **Módulo 4 – Padrões e Arquitetura**  
- ⏳ Padrões de Projeto (GoF, SOLID, MVC)  
- ⏳ Padrões Criacionais, Estruturais e Comportamentais  
- ⏳ Introdução ao Django  

### **Módulo 5 – Desenvolvimento Web com Django**  
- ⏳ Estrutura do Framework (MTV)  
- ⏳ Models, Views e Templates  
- ⏳ Django Admin  
- ⏳ Autenticação e Autorização  
- ⏳ Segurança em Aplicações Web  
- ⏳ Integração com Web Services (REST/SOAP)  

### **Módulo 6 – Projeto Final**  
- ⏳ Desenvolvimento de um projeto completo em Django  
- ⏳ Boas práticas e versionamento  
- ⏳ Apresentação do projeto (Pitch)
